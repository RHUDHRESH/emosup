# Technology Stack

## Core Technologies
- **Frontend Framework:** Next.js (React)
- **Frontend Language:** TypeScript
- **Styling:** Tailwind CSS, Framer Motion
- **Backend Framework:** Flask
- **Backend Language:** Python
- **Secondary UI:** Streamlit (for rapid prototyping/dashboard)

## AI & Machine Learning
- **Orchestration:** CrewAI (Multi-agent system)
- **Integration:** LangChain
- **Inference Engine:** Groq API (Cloud Inference) - *Replaces local Ollama*
- **Analysis:** TextBlob (Sentiment Analysis)

## Data & Storage
- **Database:** SQLite (Embedded)
- **Visualization:** Plotly, Pandas

## Infrastructure & DevOps
- **Execution:** Native Python/Node.js runtime (No Docker)
- **CI/CD:** GitHub Actions
- **Testing:** pytest
- **Linting/Formatting:** flake8, ESLint

## Security
- **Authentication:** Session-based
- **Encryption:** bcrypt (Password Hashing)
